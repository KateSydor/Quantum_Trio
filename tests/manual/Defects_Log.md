## Умовні позначення

### Пріоритет
| Рівень | Опис |
|--------|------|
| 🔴 P1 — Critical | Блокує основну функціональність, потребує негайного виправлення |
| 🟠 P2 — High | Значний вплив на UX, виправити до релізу |
| 🟡 P3 — Medium | Помірний вплив, виправити у наступному спринті |
| 🟢 P4 — Low | Мінімальний вплив, можна відкласти |

### Статус
| Статус | Опис |
|--------|------|
| 🆕 New | Нещодавно знайдений |
| 🔧 In Progress | В процесі виправлення |
| ✅ Fixed | Виправлено |
| 🔄 Retest | Потребує повторної перевірки |
| ❌ Won't Fix | Не буде виправлено |
| 🔁 Reopened | Повторно відкритий |

### Severity
| Рівень | Опис |
|--------|------|
| 🔴 Blocker | Система непрацездатна |
| 🟠 Major | Основна функція працює некоректно |
| 🟡 Minor | Другорядна функція з дефектом |
| 🟢 Trivial | Косметичний дефект |
